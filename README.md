# 📊 HR Analytics Dashboard (Power BI Project) #

# 📌 Project Overview :

This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee data and uncover key insights related to attrition, workforce distribution, and employee performance. The dashboard helps HR teams and decision-makers understand patterns in employee behavior and take data-driven actions to improve retention and productivity.

# 🎯 Objectives :
Analyze employee attrition trends
Identify factors affecting employee turnover
Monitor workforce distribution across departments
Evaluate employee demographics and job roles
Provide actionable insights for HR decision-making

# 📂 Dataset Information :
 - File Name: HR_Analytics.csv 
 - Total Records: ~1400+ employees (adjust if needed) 

# Key Columns:
 - Age
 - Gender
 - Department
 - Job Role
 - Monthly Income
 - Attrition
 - Years at Company
 - Job Satisfaction
 - Education Field
 - Work-Life Balance
   
# 🛠️ Tools & Technologies Used :
  - Power BI – Data visualization & dashboard creation
  - DAX (Data Analysis Expressions) – Measures & calculations
  - Power Query – Data cleaning and transformation
  - Excel/CSV – Data source
    
# 📊 Dashboard Features :
🔹 Key KPIs
  - Total Employees
  - Attrition Count
  - Attrition Rate (%)
  - Average Age
  - Average Salary
  - Average Years at Company
    
🔹 Visualizations Included
  - Attrition by Department
  - Attrition by Job Role
  - Attrition by Age Group
  - Gender Distribution
  - Salary Analysis
  - Years at Company vs Attrition
  - Job Satisfaction Analysis
    
🔹 Interactive Elements Slicers for:
  - Department
  - Gender
  - Education Field
  - Dynamic filtering across visuals
    
# 📈 Key Insights :
  - Higher attrition observed in specific departments (e.g., Sales/HR)
  - Employees with lower job satisfaction show higher attrition
  - Younger employees tend to leave more frequently
  - Work-life balance plays a critical role in retention
  - Salary differences impact employee turnover
